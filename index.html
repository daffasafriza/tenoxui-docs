What is TenoxUI?

TenoxUI is a Utility-First CSS Framework built for elevate web development by enhancing speed and efficiency by delivering various collection of utility-first class. Also, make it faster with our component library that you can use for your projects.

Quick Learn

Let's make your first component.

In this section you we can learn from a few code that use TenoxUI for styling your component. Let's take a look into this simple and really easy to understanding code below :

Javascript :

Make your property and styles through javascript code. 

// Make New Property You Desired            
MultiProps({

  dis: ["display"],

  "fx-par": ["justify-content", "align-items"],

});

// Define your custom styles
MultiStyles({
  
  ".card-parent": "bg-[card-bg] tc-[card-text] p-1rem br-8px dis-flex fx-par-center gap-1rem",
  
  ".card-image": "br-50% bg-[primary] box-3rem dis-flex fx-par-center",
  
  ".card-name > p.title": "fs-1rem fw-600",

  ".card-name > p.desc": "fs-14px fw-400",
  
  ".card-parent > :last-child": "ml-3rem fs-1rem",

});

CSS :

Let's add some css variable.

:root {
  --primary: #d2e603;
  --card-bg: #efefef;
  --card-text: #0d0d0d;
}

HTML :

Now, add a few line of HTML.

<div class="card-parent">
  <div class="card-image">
    <p>BG</p>
  </div>
  <div class="card-name">
    <p class="title">Some item</p>
    <p class="desc">$40</p>
  </div>
  <span>your_icon</span>
</div>

<div class="card-parent">
  <div class="card-image bg-lightblue">
    <p>BG</p>
  </div>
  <div class="card-name">
    <p class="title">Some item</p>
    <p class="desc tc-purple fw-700">$230</p>
  </div>
  <span>your_icon</span>
</div>

As you can see, you can put another style after you defined it using multiStyles function to apply the styles.
