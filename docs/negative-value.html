    How can i use negative value like -1rem or -100%?

    It's really simple question. You can add negative value like -1px or -100% by just type it directly after type. Here's a few example of the usage :

    t--10px : It will set top: -10px

    rt--45deg : It will set rotate: -45deg

    And so on. Its really simple, right? And negative value is not ocasionally used, just for a few case.
