    Padding & Margin.

    In this page, you will learn about padding and margin rules in TenoxUI. There's a few common rules to set the padding and margin.

    Padding Rule

    pt- : Set top padding

    pl- : Set left padding

    pb- : Set bottom padding

    pr- : Set right padding

    ph- : Set padding for left and right (Horizontal)

    pv- : Set padding for left and right (Vertical)

    info

    Margin Rule

    mt- : Set top margin

    ml- : Set left margin

    mb- : Set bottom margin

    mr- : Set right margin

    mh- : Set margin for left and right (Horizontal)

    mv- : Set margin for left and right (Vertical)

    Quick Learn

    Here's a simple code to apply padding

    HTML :

    <div class="bg-lightgreen pt-1rem pl-1rem">
  <div class="box-50px bg-orange"></div>
</div>
