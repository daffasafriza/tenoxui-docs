    Flexbox

    Let's make a Flexbox layout.

    Display Flex

    If you want to use flexbox, you need to make the element have flex property. And, your element will automatically have display: flex if the element have class started with flex- or fx- class.

    Align Property

    Here's all class you can add :

    ac- : Set align-content property.

    ai- : Set align-items property.

    as- : Set align-self property.

    Justify Property

    Here's all class you can add :

    jc- : Set justify-content property.

    ji- : Set justify-items property.

    js- : Set justify-self property.

    Note : You can't set value that have two words or words that separated by hypens like flex-start and space-between. So, i recommend you to use the css variable instead. Learn more about Double Value.

    Flex Property

    All type and proeprty you can add on your element :

    fd- : Set flex-direction property.

    flex- : Set flex property. FYI, flex- type have custom value, 1 1 ${value}. Example, if you use class flex-250px, it will have value for flex: 1 1 250px;

    fx-basis- : Set flex-basis property.

    fx-grow- : Set flex-grow property.

    fx-shrink- : Set flex-shrink property.

    fx-wrap- : Set flex-wrap property.
